Introduction

This is the first event in what we hope to be a series of Game AI Hackathons. To be notified about future events, join our mailing list.

The JavaScript bot included in the SDK is currently broken, a new version of the SDK will be released when it is fixed, and this notice will be updated.

SkullSpace's Game AI Hackathon is a 48 hour event during which participants write bots that compete against each other in a relatively simple game.

Starting on Friday, September 19th at Noon CDT, the SDK will be made available and the game will be announced. At 6pm the server will open, and the competition will run until Sunday, September 21st at 6pm.

The SDK contains:

  1. Example bot in C#
  2. Example bot in Java
  3. Example bot in JavaScript
  4. Example bot in Perl
  5. Example bot in PHP
  6. Example bot in Python
  7. Example bot in Ruby

The bots entered into this competition will communicate with the server over HTTP POST requests containing JSON in the request bodies. Bots may be programmed in any language. To make your life easier, each of the bots in the SDK comes fully-functional, so that if you unzip the SDK and run any bot, it is already complete enough to start playing the game on the server. All the example bots make terrible decisions, however, and it's your goal to fix it.

Participants need not be on site at the hackerspace, nor is there any requirement that participants be members. However, there will be a number of locals gathering at SkullSpace, 2nd Floor, 374 Donald Ave., from Noon to 6pm on Saturday, September 20th, 2014. This event is open to the general public. There are no prizes, only bragging rights and stories.
