API Method :: BID

You have requested this page using the GET HTTP request method, to actually invoke this API endpoint you are required to use POST.

All JSON in this page has been formatted with JSONLint, whitespace is not significant to the server. The order of keys in JSON object doesn't matter.

This method permits you to bid on the number of rounds you believe you can win in the current game you are playing, based on the cards you have been dealt. This method accepts JSON in the body of the HTTP request, as follows:

{
    "session": "A hex string will appear here."
    "bid": 5
}

The bid that you make must be an integer between 1 and 13 inclusive, as each game consists of 13 rounds. In the above example we have bid 5.


Possible responses:

  1. The server will respond with the following if your bid is accepted:

    {
    "result": "success"
}

  2. The server will respond with the following if the bidding process failed:

    {
    "result": "failure",
    "reason": "An explanation of the failure will appear here."
}

Once the bid process has completed, and once it is your turn, you have 120 seconds to play your first card, before you forfeit the game.
